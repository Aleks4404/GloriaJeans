# Test Case
## Перечень автоматизируемых сценариев:

### **1. Тестирование формы "Вход" и "Регистрация".**

**Предусловия:**
1. Запустить приложение;
2. Открыть браузер Google Chrome, ввести следующий адрес: "https://www.gloria-jeans.ru";
3. Закрыть появившейся Pop-up "Регион доставки - "город"?" нажав на кнопку "Да, верно";
4. Открыть форму "Вход" и "Регистрация" нажав на кнопку "Войти" на верхней панели справа.

_**1.1.** Регистрация с помощью Email_
1. Выполнить пункты "Предусловия"
2. Выбрать вкладку "Регистрация";
3. Выбрать radio-button "По email";
4. Поле "Имя" заполнено с помощью библиотеки генерации данных "Faker" на кириллице;
5. Поле "Email" заполнено с помощью библиотеки генерации данных "Faker";
6. Поле "Пароль" заполнено с помощью библиотеки генерации данных "Faker";
7. Нажать на кнопку "Зарегистрироваться"

**Ожидаемый результат:**
1. Регистрация проходит успешно;
2. Страница обновляется;
3. Появляется pop-up "Добро пожаловать".
4. Справа вверхнем меню страницы надпись авторизации "Войти" меняется на имя успешно зарегистрированного пользователя.

_**1.2.** Регистрация с помощью номера телефона_

_**2.11.** Добавление и удаление товара из корзины_

1. Выполнить пункты "Предусловия";
2. Нажать на слово "ЖЕНЩИНЫ" из списка на главной странице;
3. Найти из списка одежду и нажать на описание одежды например "Чёрная удлинённая куртка oversize с пером и пухом";
4. Нажать на знак "Корзина" под картинкой, чтобы добавить в избранное;
5. Выбрать размер одежды, например "L";
6. В появившемся pop-up "Товар добавлен в корзину" нажать кнопку "Продолжить покупки";
7. Нажать на иконку "Корзина", которая находится справа на вверху страницы;
8. Найти в корзине "Чёрная удлинённая куртка oversize с пером и пухом";
9. Нажать на знак "Корзина", чтобы удалить из корзины;

**Ожидаемый результат:**

1. Одежда удаляется;
2. Появляется сообщение "Оформить заказ вы сможете, как только в корзине появится товар, доступный для покупки.".

_**2.12.** Добавление в корзину и покупка одежды_

1. Выполнить пункты "Предусловия";
2. Нажать на слово "МУЖЧИНЫ" из списка на главной странице;
3. Найти из списка одежду и нажать на описание одежды например "ХАКИ СТЁГАНАЯ КУРТКА-БОМБЕР OVERSIZE";
4. Нажать на знак "Корзина" под картинкой, чтобы добавить в избранное;
5. Выбрать размер одежды, например "L/182";
6. В появившемся pop-up "Товар добавлен в корзину" нажать кнопку "Продолжить покупки";
7. Нажать на иконку "Корзина", которая находится справа на вверху страницы;
8. Найти в корзине "Чёрная удлинённая куртка oversize с пером и пухом";
9. Нажать на знак "ОФОРМИТЬ ЗАКАЗ";

**Ожидаемый результат:**

1. Страница обновляется;
2. Открывается форма "Оформление заказа" для ввода данных.

_**2.13.** Поиск и покупка одежды_

1. Выполнить пункты "Предусловия";
2. Нажать на слово "ПОДРОСТКИ" из списка на главной странице;
3. Найти из списка одежду и нажать на описание одежды например "КРАСНАЯ КУРТКА С СИНТЕПУХОМ ДЛЯ МАЛЬЧИКА";
4. Нажать на знак "Корзина" под картинкой, оформить заказ;
5. Выбрать размер одежды, например "7-8л/128";
6. В появившемся pop-up "Товар добавлен в корзину" нажать кнопку "Оформить заказ";

**Ожидаемый результат:**

1. Страница обновляется;
2. Открывается форма "Оформление заказа" для ввода данных.
